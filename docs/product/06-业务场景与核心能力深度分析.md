## 6. 三大能力壁垒

### 6.1 引用溯源闭环（CAP1+CAP2）
通用 LLM 无法做到"答案 + 引用 + 一键跳转原文 chunk"。这是 Hermes KB 的护城河，设计上必须让引用卡片**视觉权重高于普通文本**，跳转**可感知**（2000ms 高亮动画）。

### 6.2 私有知识沉淀飞轮（CAP4+CAP5+CAP7）
导入 → 组织 → 问答 → 健康度反馈 → 再导入的闭环。仪表盘的飞轮健康度是飞轮的"仪表盘"，让用户感知知识库在变好。

### 6.3 酒类垂直语境（CAP5 分类预设 + 种子知识）
预设酒类分类（六大基酒 + 中国白酒 + 鸡尾酒 + 酿造酒 + 历史文化）和种子知识，让冷启动即专业。
